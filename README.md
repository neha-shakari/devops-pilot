# DevOpsPilot 🚀

AI-powered DevOps automation platform that helps developers generate essential DevOps files using AI.

## Features

### 🐳 Dockerfile Generator
Generate production-ready Dockerfiles from project details.

### ⚙️ CI/CD Generator
Generate GitHub Actions workflows.

### ☸️ Kubernetes YAML Generator
Generate Kubernetes Deployment and Service manifests.

### 🔍 GitHub Analyzer
Analyze repositories and get DevOps recommendations.

### 🤖 AI Log Analyzer
Analyze application logs and get possible solutions.
